1. Perpustakaan Nasional Republik Indonesia

 

 Perpustakaan Nasional Republik Indonesia (Perpusnas) adalah Lembaga Pemerintah Nonkementerian yang melaksanakan tugas pemerintahan dalam bidang perpustakaan yang berfungsi sebagai perpustakaan pembina, perpustakaan rujukan, perpustakaan deposit, perpustakaan penelitian, perpustakaan pelestarian, dan pusat jejaring perpustakaan, serta berkedudukan di ibu kota negara. Gedung Fasilitas Layanan Perpustakaan Nasional berlokasi di Jalan Medan Merdeka Selatan No.11, Jakarta dan sebagian besar perkantorannya di Jalan Salemba Raya No. 28A. Perpustakaan Nasional berada di bawah dan bertanggung jawab kepada Presiden.

 Berikut tautan:Perpustakaan Nasional

2. Perpustakaan Universitas Sumatera Utara

 

 Perpustakaan Universitas Sumatera Utara (USU) merupakan sebuah lembaga yang menyediakan layanan informasi dan sumber belajar bagi mahasiswa, dosen, peneliti, dan masyarakat umum. Terletak di kampus Universitas Sumatera Utara, perpustakaan ini memiliki koleksi buku, jurnal ilmiah, tesis, disertasi, dan sumber informasi lainnya yang luas dan beragam. Layanan yang disediakan meliputi pinjam-meminjam buku, konsultasi referensi, akses ke sumber elektronik, serta kegiatan pendukung pembelajaran dan penelitian seperti pelatihan literasi informasi. Perpustakaan USU bertujuan untuk mendukung pengembangan akademik dan riset di lingkungan universitas serta memfasilitasi akses terhadap informasi bagi masyarakat luas.

 Berikut tautan:Perpustakaan Universitas Sumatera Utara

3. Perpustakaan Kantor Perwakilan Bank Indonesia Provinsi Sumatera Utara

 

 Perpustakaan di Kantor Perwakilan Bank Indonesia Provinsi Sumatera Utara adalah sebuah fasilitas yang menyediakan sumber daya informasi terkait dengan bidang keuangan, ekonomi, dan perbankan. Di sini, Anda dapat menemukan berbagai macam literatur, jurnal, laporan, dan publikasi terkait dengan kebijakan moneter, stabilitas keuangan, statistik ekonomi, serta topik-topik terkait lainnya. Perpustakaan ini merupakan salah satu sarana untuk mendukung kegiatan riset, analisis, dan pengambilan keputusan di lingkungan Bank Indonesia, baik oleh staf internal maupun masyarakat umum yang membutuhkan informasi terkait sektor keuangan dan ekonomi di Provinsi Sumatera Utara.

Pengenalan HTML dan CSS

Pengenalan HTML dan CSS

Pengertian HTML

HTML adalah bahasa markup untuk membuat halaman web. Kepanjangan dari HTML adalah Hypertext Markup Language. Artinya, HTML menggunakan simbol tertentu (tag) yang nantinya akan diterjemahkan oleh browser ke halaman web.HTML terdiri dari kombinasi teks dan simbol yang disimpan dalam sebuah file. Untuk membuat file HTML, harus mengikuti standar atau format khusus. Format ini tertuang dalam standar kode internasional atau ASCII (American Standard Code for Information Interchange).

Fungsi HTML

  • Membuat struktur halaman web
  • Menambahkan konten website
  • mengatur format dan tata letak
  • mengarahkan pengguna ke halaman website lain

Pengertian CSS

Cascading Style Sheets (CSS) merupakan bahasa pemrograman yang digunakan untuk menentukan bagaimana dokumen dan website akan disajikan. CSS dibuat oleh Word Wide Web Consortium (W3C) pada 1996. CSS merupakan kode-kode yang dipakai untuk mendesain laman HTML.adi, CSS berisi kumpulan perintah yang digunakan untuk menjelaskan tampilan halaman situs web dalam mark-up language, seperti HTML yang terkenal sebagai bahasa pemrograman standar dan sering digunakan dalam proses pembuatan website. CSS hadir sebagai pemisah konten dari tampilan visualnya di situs di mana tentunya berpengaruh pada tampilan sebuah website.

Fungsi CSS

  • Proses loading halaman web lebih cepat
  • Beragam variasi
  • Tampilan website akan lebih rapi
